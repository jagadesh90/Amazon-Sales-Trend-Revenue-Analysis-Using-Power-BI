# Amazon Sales Analysis Dashboard – Power BI
## Project Overview

-This project presents a comprehensive Amazon Sales Analysis Dashboard developed using Power BI. The dashboard provides actionable business insights by analyzing revenue, profit, orders, product categories, customer regions, discounts, and ratings.
-The goal of this project is to support data-driven decision-making through interactive visualizations and KPI tracking.
 
# Business Objectives

- Analyze overall sales and profitability performance
- Identify top-performing product categories
- Evaluate regional sales distribution
- Understand the impact of discounts on revenue and profit
- Examine customer ratings and review trends
- Monitor monthly revenue and profit trends

# Key Performance Indicators (KPIs)

- Total Revenue: 32.87M
-  Total Profit: 1.64M
- Total Orders: 50K
- Total Quantity Sold: 150K
- Average Customer Rating: 3.00

# Dashboard Features
# Executive Overview Page

- Revenue Trend Over Time
- Profit Trend Over Time
- Revenue by Product Category
- Revenue by Customer Region (Map Visualization)
- Revenue by Payment Method
- Interactive Filters (Category, Date, Region, Review Count)

# Deep Analysis Page

- Profit by Category
- Revenue vs Profit Comparison
- Rating vs Revenue Analysis
- Review Count vs Revenue Analysis
- Discount Percentage vs Revenue
- Discount Percentage vs Profit
- Category Performance by Region
